## Product Shop
POC for a product shop using NestJS and MongoDB

## Technologies
- NestJS
- MongoDB
- Docker
- Docker Compose
- Mongo Express
- JWT
- Mongoose

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
